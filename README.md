# Student Data Management System (C)

## Overview

Student Data Management System is a console-based application developed in C that helps manage student records using file handling. The system allows users to add new students, view existing student records, and remove student information from different classes. Student data is stored in text files, ensuring persistence even after the program is closed.

This project was developed as an academic project to demonstrate concepts such as file handling, menu-driven programming, conditional statements, loops, functions, and basic data management.

## Features

- Add new student records
- View all student records
- Remove student records
- Separate records for:
  - F.Y.B.Sc.IT
  - S.Y.B.Sc.IT
  - T.Y.B.Sc.IT
- File-based storage using text files
- Interactive menu-driven interface
- Persistent data storage

## Technologies Used

- C Programming Language
- File Handling (`fopen`, `fprintf`, `fgets`, `fclose`)
- Standard Input/Output Operations
- Menu-Driven Programming

## How It Works

1. Launch the application.
2. Select an option from the main menu.
3. Add, view, or remove student records.
4. Student information is stored in text files according to the selected class.
5. Data remains saved for future use.
